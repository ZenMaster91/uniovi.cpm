Find cruises

On this window you can find all cruises are available.

By default just open the application displays all available cruises

To filter the different cruises on the left sidebar you will find a series of filters. You can filter by: Location, Start date, departure port, whether to accept kids or not and if they are discount or not.

Besides you can sort the list of cruises: Duration, Start Date or ascending price. To use any filter just edit it and automatically apply to the list of cruises.

Warning window: No cruises are available for search.

Sometimes you may receive an error window with the title above. Do not worry just click accept.

To select any cruise you only have to click button "Buy Now"

Sometimes you can reach a point where there are not avaliable creuises. Don worry, just try another search.



Buscar cruceros

En esta ventana podrás encontrar todos los cruceros que se encuentran disponibles.

Por defecto nada más abrir la aplicación aparecerán todos los cruceros disponibles.

Para filtrar los distintos cruceros en la barra lateral izquierda encontrará una serie de filtros. Puede filtrar por: Destino, Fecha de inicio, Puerto de salida, si acepta niños o no y por si tienen descuento o no.

A parte puede ordenar la lista de cruceros por: Duración, Fecha de inicio o precio ascendente. Para utilizar cualquier filtro sólo tienes que modificarlo y automaticamente se aplicará a la lista de cruceros.

Ventana de Aviso: No se encuentran cruceros disponibles para su búsqueda

En ocasiones puede aparecer una ventana de error con el título anterior. No se preocupe tan sólo presione acceptar.

Para seleccionar cualquier crucero tan sólo tiene que hacer click en el botón de "Comprar"
