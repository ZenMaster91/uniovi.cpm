Set a reservation

On this window you can set your book, add, modify or delete cabins, passengers and extras.

To return to the previous screen in the upper left corner you will find a button that says "go back" (1). Remember if go back to search again for a reservationthe rooms in the reservation will be deleted.

Then the parts of the window are listed.



  1. Back to the search page.
  2. Image associated with the selected boat cruise.
  3. Short description of the cruise that includes the name, duration and name of the boat.
  4. Descriptive text about the selected cruise.
  5. State of the cruise. It displays the number of free rooms for each selected date (6).
  6. Select a date shall be the date on which performs the cruise. Departure date.
  7. Once a departure date you selected this button It allows change.
  8. Select the type of room inside possibilities.
  9. Passenger list associated with the room.
  10. Group of buttons for passengers. You can add, modify or remove.
  11. Panel extras. All possible extras are displayed. Select those you want.Note: The Extra beds can only be used by children under 16 years.
  12. Add the room set to the reserve.Note: Every room should have at least one older than 16 years to be added to the reserve.
  13. List of rooms on the reservation.
  14. Advance to the screen purchase summary.


Configurar una reserva

En esta ventana podrá configurar su reserva, añadir, modificar o eliminar camarotes, pasajeros y extras.

Para volver a la pantalla anterior, en la esquina superior izquierda encontrará un botón que pone "vovler a búsqueda" (1). Recuerda wue si vuelve a búsqueda durante una reserva las habitaciones seleccionadas serán eliminadas

A continuación se detallan las distintas partes de la ventana.



  1. Volver a la página de búsqueda.
  2. Imágen del barco asociado al crucero seleccionado.
  3. Breve descripción del crucero que incluye la denominación, la duración y el nombre del barco.
  4. Texto descriptivo sobre el crucero seleccionado.
  5. Estado del crucero. Muestra el número de habitaciones libres para cada fecha seleccionada (6).
  6. Seleccione una fecha, será la fecha en la que realizará el crucero. Fecha de salida.
  7. Una vez seleccionada una fecha de salida este botón te permite cambiarla.
  8. Permite seleccionar el tipo de habitación dentro de las posibilidades.
  9. Lista de pasajeros asociada a la habitación.
  10. Grupo de botones para pasajeros. Puede añadir, modificar o eliminar.
  11. Panel de extras. Se mostrarán todos los extras posibles. Seleccione aquellos que deseas. Nota: Las camas extra sólo podrán ser utilizadas por niños menores de 16 años.
  12. Añade la habitación configurada a la reserva. Nota: Toda habitación deberá tener al menos un mayor de 16 años para poder ser añadida a la reserva.
  13. Lista de habitaciones en la reserva.
  14. Avance a la pantalla de resumen de compra.
